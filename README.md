# Artificial_Neural_Network_-ANN-

🧠 Artificial Neural Network (ANN) Project
📌 Overview

This project demonstrates the implementation of an Artificial Neural Network (ANN) for solving a supervised learning problem. It covers the complete ML pipeline including data loading, preprocessing, model building, training, evaluation, and prediction.

The goal is to build a strong foundation in deep learning by understanding how neural networks learn patterns from data.

🚀 Features
End-to-end ANN pipeline
Data preprocessing and feature scaling
Model building using deep learning frameworks
Training and evaluation
Performance metrics visualization
Modular and scalable project structure
